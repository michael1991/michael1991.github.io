Git 使用简易指南

1、创建新仓库:

创建新文件夹->打开  git init

2、检出仓库

git clone + 地址/路径

3、添加与提交

添加（缓存区）：

git add <filename>  /  git add *

提交（HEAD）：

git commit -m "代码提交信息"

4、推送与改动

推送（分支，默认master）：

git push origin master(master可以替换成想推送的分支)

改动：

git remote add origin <server>

5、分支

创建分支：

git checkout -b feature_x (feature_x：分支名字)

切换到主分支：

git checkout master

删除分支：

git branch -d feature_x

将分支推送：

git push origin <branch>

6、更新与合并

将本地仓库更新至最新改动：

git pull

合并其他分支到你的当前分支，如master：

git merge <branch>

查看差异：

git diss <source_branch> <target_branch>

7、标签

创建一个叫做1.0.0的标签：

git tag 1.0.0 1b2e1d63ff  (1b2e1d63ff是你想要标记的提交ID的前10位自负）

获取提交ID：

git log

8、替换本地改动

git checkout -- <filename>

9、丢弃所有本地改动：

git fetch origin
git reset --hard origin/master

10、显示历史记录时，只显示一行注释信息：

git config format.pretty oneline

11、交互的添加文件至缓存区

git add -i

12、显示提交历史详细

git log -p

13、将当前改动提交到缓存区

git stash

14、从缓存区取出第一个

git stash pop
